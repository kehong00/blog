6. 代码块

作为程序员插入代码是必不可少的,插入代码的方式有两种,一种是利用tab键缩进,另一种是利用`符号包含代码

1.使用tab缩进

在代码前面加上一个tab制表符即可

public static void main(String args[]){

}

2.使用`符号包含代码

单行代码:使用一个`包含代码

System.out.println("Hello world");

多行代码块:使用```包含代码,可以在开始的```后声明使用的语言,可以自动实现语法高亮

7. 列表

1.无序列表

使用一个+,-,*都可以表示无序列表,符号后面需要有一个空格

  • 无序列表3

2.有序列表

在数字序号后面加上一个.就表示有序列表,同意需要在.后加上一个空格

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3
